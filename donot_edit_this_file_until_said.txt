Lidar

1. to start the driver
source /opt/ros/humble/setup.bash
/opt/ydlidar_ros2_driver/lib/ydlidar_ros2_driver/ydlidar_ros2_driver_node --ros-args --params-file /opt/ydlidar_ros2_driver/share/ydlidar_ros2_driver/params/X2.yaml


2. to start the scan_listener (u can uses this to test the lidar data)
cd ~/paint_ws
source /opt/ros/humble/setup.bash
source install/setup.bash
ros2 run lidar_scan_viewer scan_listener


Mavros

source /opt/ros/humble/setup.bash
ros2 launch mavros apm.launch fcu_url:=/dev/ttyACM0:57600


Obstacle avoidance

cd ~/paint_ws
source /opt/ros/humble/setup.bash
source install/setup.bash
ros2 run obstacle_avoidance avoider
