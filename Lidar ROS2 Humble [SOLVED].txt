2. Cloning LiDAR Driver – Initial Mistake
Command (Wrong):
cd ~/lidar_ws/src
git clone https://github.com/YDLIDAR/ydlidar_ros2_driver.git


Problem:
* This clones the default branch, which may not be compatible with ROS 2 Humble.
* Attempting to build caused errors.
Reference that helped:
* StackOverflow: Unable to build ydlidar_ros2 package in ROS 2 Humble on Ubuntu 22.04
Solution:
Clone the Humble branch explicitly:
git clone -b humble https://github.com/YDLIDAR/ydlidar_ros2_driver.git


* Ensures compatibility with ROS 2 Humble.
* Avoids build errors.

7. First Issue – QoS Warning
Problem: On running lidar_node:
[WARN] [lidar_reader]: New publisher discovered on topic '/scan', offering incompatible QoS. No messages will be received from it. Last incompatible policy: RELIABILITY


Cause: Subscriber QoS in lidar_node.py was set to RELIABLE, but YDLIDAR publishes with BEST_EFFORT.
Solution: Update subscriber QoS:
from rclpy.qos import QoSProfile, ReliabilityPolicy, HistoryPolicy


qos = QoSProfile(
    reliability=ReliabilityPolicy.BEST_EFFORT,
    history=HistoryPolicy.KEEP_LAST,
    depth=10
)


Result: Warning disappeared; messages received successfully.
________________


8. Second Issue – Front Distance Wrong
Problem: Node printed only left and right distances; front distance was incorrect.
Cause: ranges[0] is not guaranteed to be the front. The front is usually the middle index of the ranges array.
Solution: Calculate indices dynamically:
total_points = len(msg.ranges)
front_index = total_points // 2
left_index = total_points // 4
right_index = 3 * total_points // 4


Result: Node prints Front, Left, and Right distances correctly.
________________


9. Corrected Python Node
File: lidar_node.py
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import LaserScan
from rclpy.qos import QoSProfile, ReliabilityPolicy, HistoryPolicy


class LidarReader(Node):
    def __init__(self):
        super().__init__('lidar_reader')


        # QoS settings to match 2D LiDAR driver
        qos = QoSProfile(
            reliability=ReliabilityPolicy.BEST_EFFORT,
            history=HistoryPolicy.KEEP_LAST,
            depth=10
        )


        self.subscriber = self.create_subscription(
            LaserScan,
            '/scan',
            self.callback,
            qos
        )


    def callback(self, msg):
        total_points = len(msg.ranges)
        if total_points == 0:
            return


        front_index = total_points // 2
        left_index = total_points // 4
        right_index = 3 * total_points // 4


        front_distance = msg.ranges[front_index]
        left_distance = msg.ranges[left_index]
        right_distance = msg.ranges[right_index]


        print(f"Front: {front_distance:.2f} m | Left: {left_distance:.2f} m | Right: {right_distance:.2f} m")


def main(args=None):
    rclpy.init(args=args)
    node = LidarReader()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()


if __name__ == "__main__":
    main()
